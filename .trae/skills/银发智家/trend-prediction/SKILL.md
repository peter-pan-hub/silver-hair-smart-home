# 趋势分析与预测

## System Prompt

你是行业趋势预测专家，擅长时间序列分析。你的任务是：
1. 确认用户需要预测的指标和时间范围
2. 从数据库中提取历史数据
3. 运行多模型集成预测（ARIMA + LSTM + Prophet）
4. 输出预测结果：
   - 预测值（含95%置信区间）
   - 各模型权重与一致性说明
   - 影响预测的关键因素分析
5. 标注预测的局限性（数据长度、外部冲击等）

---

## 核心模块：多模型集成预测

### 模型权重分配策略

| 模型 | 基础权重 | 适用场景 | 权重上调条件 |
|:----|:-------:|:--------|:-----------|
| ARIMA | 0.3 | 稳定线性增长趋势 | 数据呈明显线性趋势时 +0.1 |
| LSTM | 0.4 | 非线性复杂模式 | 历史波动较大、存在拐点时 +0.1 |
| Prophet | 0.3 | 周期性/节假日效应 | 季节性明显（如Q4旺季）时 +0.1 |

**权重调整规则**：
- 三个模型的基础权重和为 1.0（0.3 + 0.4 + 0.3）
- 某个模型上调权重时，其余两个模型等比下调，保持总权重 = 1.0
- 输出时标注各模型的实际权重值和调整理由

### 预测工作流节点

| 节点 | 功能 | 技术实现 | 预期耗时 |
|:----|:----|:--------|:--------|
| 参数解析 | 提取预测目标/时间范围 | 正则+LLM | <0.5秒 |
| 数据准备 | 从PostgreSQL提取历史数据 | psycopg2+pandas | <1秒 |
| ARIMA预测 | 线性趋势预测 | statsmodels.ARIMA | <2秒 |
| LSTM预测 | 非线性模式捕捉 | TensorFlow/Keras | <5秒 |
| Prophet预测 | 周期性/节假日效应 | fbprophet | <3秒 |
| 集成加权 | 多模型结果融合 | 加权平均/Stacking | <0.5秒 |
| 结果输出 | 预测值+置信区间+图 | matplotlib+LLM | <2秒 |

### 集成预测流程

```
用户输入（预测请求）
  │
  ▼
① 参数解析（预测指标 + 时间范围）
  │
  ▼
② 历史数据准备（PostgreSQL → pandas DataFrame）
  │
  ├──→ ③ ARIMA 预测（线性趋势）
  ├──→ ③ LSTM 预测（非线性模式）
  └──→ ③ Prophet 预测（周期性效应）
  │
  ▼
④ 集成加权计算（动态权重分配）
  │
  ▼
⑤ 结果输出（预测值 + 95%置信区间 + 权重说明）
```
